# Algorithm

This project consists of a collection of popular algorithms implemented in JavaScript:

1. **Deep Copy**
2. **Recursion**
3. **Search Algorithms**
   - Binary Search
   - Linear Search
4. **Shortcut Tables**
5. **Sort Algorithms**
   - Bubble Sort
   - Heap Sort
   - Insertion Sort
   - Merge Sort
   - Quick Sort
   - Selection Sort
6. **Tree**
   - Binary Tree
   - Binary Search Tree (BST)
   - Balanced Binary Search Tree (AVL)
   - Red Black Tree
7. **Tree Algorithms**
   - Breadth First Search (BFS)
   - Depth First Search (DFS)
